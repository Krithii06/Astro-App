# Nadi Jyotish — Secrets of Nadi Astrology

A comprehensive React web application based on S. Ravi's "Secrets of Nadi Astrology".

## Features
- **South Indian Birth Chart** — Interactive Rasi chart display
- **Love & Marriage** — Timing, nature of spouse, pre-marital analysis
- **Medical Astrology** — Disease predictions by planetary afflictions
- **Career** — Profession guidance from Saturn aspects
- **Foreign Travel** — Moon+Rahu+planet combinations
- **Parents** — Sun/Moon relationship analysis
- **Dasha Periods** — Vimshottari Mahadasha & Bhukti timeline
- **Yogas** — Dhana, Sanyasa, Moksha, and all key Nadi yogas

## Deploy to Vercel
1. Push this folder to a GitHub repository
2. Go to vercel.com → New Project → Import your repo
3. Vercel auto-detects Vite — click Deploy
4. Done! Your site will be live at `your-project.vercel.app`

## Local Development
```bash

npm run dev
```
